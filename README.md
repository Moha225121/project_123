# عرض مشروع التخرج

موقع عرض ثابت بنمط شرائح أكاديمية لمشروع: **تأثير مصدر الركام الخشن على مقاومة الخرسانة للانضغاط**.

## التشغيل

افتح `index.html` مباشرة في المتصفح. لا يحتاج المشروع إلى Backend أو Node.js أو قاعدة بيانات.

## هيكل الملفات

- `index.html`: الشرائح والواجهة الرئيسية.
- `css/style.css`: نظام التصميم والتخطيط.
- `css/animations.css`: الحركات المختصرة.
- `css/print.css`: إعدادات الطباعة وPDF.
- `js/data.js`: بيانات الرسوم والجداول وملاحظات المتحدث.
- `js/charts.js`: رسوم SVG محلية قابلة للتصدير.
- `js/app.js`: التنقل، البحث، الملاحظات، المؤقت، الجداول.
- `assets/images`: الصور المستخرجة من ملف Word.
- `extracted-project-data.md`: ملخص البيانات العلمية المستخدمة.

## اختصارات لوحة المفاتيح

- السهم الأيمن، السهم الأسفل، Space، Page Down: الشريحة التالية.
- السهم الأيسر، السهم الأعلى، Page Up: الشريحة السابقة.
- Home: أول شريحة.
- End: آخر شريحة.
- F: وضع ملء الشاشة.
- O: عرض النظرة العامة.
- N: ملاحظات المتحدث.
- R: إعادة العرض والمؤقت إلى البداية.
- Escape: إغلاق النوافذ المنبثقة أو النظرة العامة.

## تعديل الأسماء والشعار

عدّل أسماء الطلبة أو عناصر الغلاف من داخل الشريحة الأولى في `index.html`. الغلاف الحالي يستخدم البيانات الظاهرة في صورة الغلاف: جامعة بلاغراي للعلوم الحديثة، كلية الهندسة وعلوم الحاسوب، قسم الهندسة المدنية / شعبة الإنشاءات، د. عائشة إبراهيم الزناتي، خريف 2025/2026م.

لاستبدال شعار الجامعة، ضع صورة الشعار داخل `assets/logo` واستبدل عنصر `.logo-placeholder` في الشريحة الأولى بعنصر `img`.

## تعديل النتائج والرسوم

كل بيانات الرسوم والجداول الرئيسية موجودة في `js/data.js`.

عدّل المصفوفات مثل `gradationData`, `crushingData`, `impactData`, و`comparisonRows`، ثم افتح الصفحة من جديد.

## الطباعة إلى PDF

استخدم زر `PDF` داخل العرض أو اختصار الطباعة في المتصفح. كل شريحة تُطبع في صفحة منفصلة بنمط أفقي 16:9 قدر الإمكان.

## استبدال الصور المختبرية

الصور الحالية مستخرجة من ملف DOCX الأصلي، لكنها غير مصنفة بدقة آلية. يمكن استبدالها يدويًا داخل `assets/images` وتحديث مساراتها في الشريحة 7 أو الشريحة الختامية.

## تحذير علمي

لم تُخترع نتائج مقاومة ضغط أو هبوط. يعرض الموقع بوضوح أن نتائج الضغط والهبوط غير متوفرة في فصل النتائج، وأن أجدابيا أفضل فقط في اختباري التهشيم والصدمية حسب البيانات المتاحة.

## التوافق

اختُبر التصميم ليعمل في المتصفحات الحديثة مثل Chrome وEdge وFirefox. يوصى باستخدام Chrome أو Edge عند العرض على Projector أو عند تصدير الرسوم إلى PNG.
